# ReactNative with TSX and Expo-CLI

## Available Scripts
### Development
- `yarn start` # you can open iOS, Android, or web from here, or run them directly with the commands below.
- `yarn android`
- `yarn ios`
- `yarn web`

### Production
- `expo publish`
- `expo build`

#### Deploy to Vercel
- `expo build:web`
- `cd web-build`
- run `vercel`
